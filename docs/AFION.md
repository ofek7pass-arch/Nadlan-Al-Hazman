## מטרת הפרויקט
סוכן אוטומטי שסורק מקורות נדל"ן מרובים ומסנן דירות לפי קריטריונים אישיים. מספק התראות בזמן אמת כשדירה רלוונטית מופיעה. **הממשק הסופי חייב להיות לייב (לא רק מקומי).**

---

## החלטות שהתקבלו בשיחה זו

### פייסבוק — נדחה לשלב מאוחר
- Facebook חוסמת scrapers מ-2018
- CrowdTangle נסגר אוגוסט 2024, המחליף (Meta Content Library) — אקדמיה בלבד
- חלופות מסחריות (Brandwatch) — $1,000-5,000+/חודש
- **החלטה:** מתחילים בלי Facebook. חוזרים אליו בשלב מאוחר אם צריך.
- **מסמך מלא:** `docs/DECISIONS.md`

### Deploy — Railway
- תקדים עובד מ-ai-personal-agent
- הוספת פרויקט חדש ב-Railway = 3 קליקים, כל פרויקט עצמאי
- **עלות:** חינמי (tier ראשון) → $5/חודש אחרי 512MB RAM

## מפת מקורות נתונים

### קטגוריה 1 — ירוק (קל, מיידי)
| מקור | כלי | הערות |
|------|-----|-------|
| **יד2** | `yad2.js` (npm package קיים!) | ~70% מהשוק, כלים מוכנים |
| **Telegram** | `telegraf` (npm) | API פתוח, חינמי, גם להתראות |
| **homeless.co.il** | `cheerio` | HTML פשוט, אנטי-בוט מינימלי |
| **nadlan.com** | `cheerio` | HTML סטטי ישן |

### קטגוריה 2 — כתום (בינוני, אפשרי)
| מקור | כלי | הערות |
|------|-----|-------|
| **מדלן** | Playwright + Apify scraper | JavaScript SPA, כלים קיימים |
| **winwin.co.il** | Playwright | אין כלים מוכנים |
| **komo.co.il** | Playwright | אין כלים מוכנים |

### קטגוריה 3 — אדום (לא רלוונטי)
| מקור | סיבה |
|------|------|
| **dira.moch.gov.il** | מערכת הגרלות, לא מודעות חיות |
| **Facebook** | חסום, יקר, נגד ToS |

### בונוס
- **data.gov.il** — API רשמי ממשלתי, נתוני מחירים היסטוריים לפי שכונה (לא מודעות חיות)

## סטאק טכנולוגי שנבחר
- **שפה:** Node.js (עקביות עם ai-personal-agent)
- **סריקה:** `yad2.js` + `cheerio` + `playwright`
- **התראות:** Telegram Bot (`telegraf`)
- **Scheduler:** `node-cron`
- **DB:** SQLite → PostgreSQL בהמשך
- **Deploy:** Railway
